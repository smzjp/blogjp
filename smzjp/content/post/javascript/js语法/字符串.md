### 字符串方法



1. split

   字符串分割,返回数组

   ```javascript
   let str = 'aaa,bbb,ccc'
   let arr = str.split(',')
   console.log(arr)
   ```

2. substring

   字符截取，根据下标

   ```javascript
   let res = str.substring(4)
   console.log(res)
   res = str.substring(4,7)
   console.log(res)
   ```
